# dataio Package Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the `dataio` package providing CSV, JSON, and Parquet readers/writers for gopandas DataFrames, with automatic type inference for CSV and multiple orientation formats for JSON.

**Architecture:** Package `dataio/` with sub-packages `dataio/csv`, `dataio/json`, `dataio/parquet`. Each provides `From*()` reader functions returning `dataframe.DataFrame` and `To*()` writer functions accepting a DataFrame. CSV uses `encoding/csv` stdlib with a functional options pattern and automatic type inference (int64 → float64 → string). JSON uses `encoding/json` with three orient modes. Parquet uses `github.com/apache/arrow-go/v18/parquet/pqarrow` for zero-copy Arrow interop.

**Tech Stack:** Go 1.21+, github.com/apache/arrow-go/v18, encoding/csv, encoding/json

**Depends on:** dtype, index, arrowutil, series, dataframe (must be implemented first)

```go
package csv

import (
	"github.com/vchitepu/gopandas/lib/dtype"
)

// csvConfig holds all configuration for CSV reading/writing.
type csvConfig struct {
	sep           rune
	header        bool
	indexCol      string
	useCols       []string
	naValues      map[string]bool
	nRows         int // 0 means read all
	skipRows      int
	dtypeOverride map[string]dtype.DType
}

func defaultConfig() csvConfig {
	return csvConfig{
		sep:           ',',
		header:        true,
		naValues:      map[string]bool{"": true, "NA": true, "NaN": true, "null": true, "<NA>": true},
		dtypeOverride: make(map[string]dtype.DType),
	}
}

// CSVOption is a functional option for CSV operations.
type CSVOption func(*csvConfig)

// WithSep sets the field separator (default: ',').
func WithSep(sep rune) CSVOption {
	return func(c *csvConfig) {
		c.sep = sep
	}
}

// WithHeader controls whether the first row is a header (default: true).
func WithHeader(has bool) CSVOption {
	return func(c *csvConfig) {
		c.header = has
	}
}

// WithIndexCol sets which column to use as the index.
func WithIndexCol(col string) CSVOption {
	return func(c *csvConfig) {
		c.indexCol = col
	}
}

// WithUseCols limits which columns to read.
func WithUseCols(cols []string) CSVOption {
	return func(c *csvConfig) {
		c.useCols = cols
	}
}

// WithNAValues sets the strings to interpret as NA/null.
func WithNAValues(vals []string) CSVOption {
	return func(c *csvConfig) {
		c.naValues = make(map[string]bool, len(vals))
		for _, v := range vals {
			c.naValues[v] = true
		}
	}
}

// WithNRows limits the number of data rows to read (0 = all).
func WithNRows(n int) CSVOption {
	return func(c *csvConfig) {
		c.nRows = n
	}
}

// WithSkipRows skips the first n data rows (after header).
func WithSkipRows(n int) CSVOption {
	return func(c *csvConfig) {
		c.skipRows = n
	}
}

// WithDTypeOverride forces a column to a specific dtype instead of inferring.
func WithDTypeOverride(col string, d dtype.DType) CSVOption {
	return func(c *csvConfig) {
		c.dtypeOverride[col] = d
	}
}
```

```go
package csv

import (
	"encoding/csv"
	"fmt"
	"io"
	"strconv"

	"github.com/apache/arrow-go/v18/arrow"
	"github.com/apache/arrow-go/v18/arrow/array"
	"github.com/apache/arrow-go/v18/arrow/memory"
	"github.com/vchitepu/gopandas/lib/dataframe"
	"github.com/vchitepu/gopandas/lib/dtype"
	"github.com/vchitepu/gopandas/lib/index"
)

// FromCSV reads CSV data from r and returns a DataFrame.
// Column types are inferred automatically: int64 → float64 → string.
// Use CSVOption functions to customize parsing behavior.
func FromCSV(r io.Reader, opts ...CSVOption) (dataframe.DataFrame, error) {
	cfg := defaultConfig()
	for _, opt := range opts {
		opt(&cfg)
	}

	reader := csv.NewReader(r)
	reader.Comma = cfg.sep
	reader.LazyQuotes = true

	// Read all records
	allRecords, err := reader.ReadAll()
	if err != nil {
		return dataframe.DataFrame{}, fmt.Errorf("csv read: %w", err)
	}
	if len(allRecords) == 0 {
		return dataframe.DataFrame{}, fmt.Errorf("csv: empty input")
	}

	// Extract header and data rows
	var colNames []string
	var dataRows [][]string

	if cfg.header {
		colNames = allRecords[0]
		dataRows = allRecords[1:]
	} else {
		// Auto-generate column names: "0", "1", "2", ...
		nCols := len(allRecords[0])
		colNames = make([]string, nCols)
		for i := 0; i < nCols; i++ {
			colNames[i] = strconv.Itoa(i)
		}
		dataRows = allRecords
	}

	// Apply skipRows
	if cfg.skipRows > 0 {
		if cfg.skipRows >= len(dataRows) {
			dataRows = nil
		} else {
			dataRows = dataRows[cfg.skipRows:]
		}
	}

	// Apply nRows
	if cfg.nRows > 0 && cfg.nRows < len(dataRows) {
		dataRows = dataRows[:cfg.nRows]
	}

	// Build column index map for useCols filtering
	colIndices := make([]int, 0, len(colNames))
	selectedNames := make([]string, 0, len(colNames))

	if len(cfg.useCols) > 0 {
		useSet := make(map[string]bool, len(cfg.useCols))
		for _, c := range cfg.useCols {
			useSet[c] = true
		}
		for i, name := range colNames {
			if useSet[name] {
				colIndices = append(colIndices, i)
				selectedNames = append(selectedNames, name)
			}
		}
	} else {
		for i, name := range colNames {
			colIndices = append(colIndices, i)
			selectedNames = append(selectedNames, name)
		}
	}

	// Extract raw string columns
	nRows := len(dataRows)
	rawCols := make([][]string, len(colIndices))
	for i := range rawCols {
		rawCols[i] = make([]string, nRows)
	}
	for rowIdx, row := range dataRows {
		for colIdx, srcIdx := range colIndices {
			if srcIdx < len(row) {
				rawCols[colIdx][rowIdx] = row[srcIdx]
			}
		}
	}

	// Handle indexCol — find it, extract it, remove from data columns
	var dfIndex index.Index
	var indexColIdx int = -1

	if cfg.indexCol != "" {
		for i, name := range selectedNames {
			if name == cfg.indexCol {
				indexColIdx = i
				break
			}
		}
		if indexColIdx == -1 {
			return dataframe.DataFrame{}, fmt.Errorf("csv: index column %q not found", cfg.indexCol)
		}

		// Build the index from the index column values
		indexLabels := rawCols[indexColIdx]
		dfIndex = index.NewStringIndex(indexLabels, cfg.indexCol)

		// Remove index column from data
		newNames := make([]string, 0, len(selectedNames)-1)
		newCols := make([][]string, 0, len(rawCols)-1)
		for i := range selectedNames {
			if i != indexColIdx {
				newNames = append(newNames, selectedNames[i])
				newCols = append(newCols, rawCols[i])
			}
		}
		selectedNames = newNames
		rawCols = newCols
	} else {
		dfIndex = index.NewRangeIndex(nRows, "")
	}

	// Infer types and build Arrow arrays for each column
	alloc := memory.DefaultAllocator
	fields := make([]arrow.Field, len(selectedNames))
	arrays := make([]arrow.Array, len(selectedNames))

	for i, name := range selectedNames {
		col := rawCols[i]

		// Check for dtype override
		if override, ok := cfg.dtypeOverride[name]; ok {
			arr, err := buildColumnWithDType(alloc, col, override, cfg.naValues)
			if err != nil {
				return dataframe.DataFrame{}, fmt.Errorf("csv: column %q: %w", name, err)
			}
			arrowDT, _ := dtype.DTypeToArrow(override)
			fields[i] = arrow.Field{Name: name, Type: arrowDT, Nullable: true}
			arrays[i] = arr
			continue
		}

		// Auto-infer type: try int64, then float64, fall back to string
		inferredDType, arr := inferColumn(alloc, col, cfg.naValues)
		arrowDT, _ := dtype.DTypeToArrow(inferredDType)
		fields[i] = arrow.Field{Name: name, Type: arrowDT, Nullable: true}
		arrays[i] = arr
	}

	// Build Arrow Record and convert to DataFrame
	schema := arrow.NewSchema(fields, nil)
	rec := array.NewRecord(schema, arrays, int64(nRows))
	defer rec.Release()

	// Release individual arrays (Record retains them)
	for _, arr := range arrays {
		arr.Release()
	}

	df := dataframe.FromArrow(rec)

	// If we have a custom index, we need to set it
	if cfg.indexCol != "" {
		_ = dfIndex // The index is already set via the index column
		// We need to reconstruct with the proper index
		df = dataframe.FromArrowWithIndex(rec, dfIndex)
	}

	return df, nil
}

// inferColumn tries to parse all values in a column as int64, then float64,
// falling back to string. Returns the inferred DType and the built Arrow array.
func inferColumn(alloc memory.Allocator, values []string, naValues map[string]bool) (dtype.DType, arrow.Array) {
	// Pass 1: try int64
	int64Vals := make([]int64, len(values))
	nulls := make([]bool, len(values)) // true = null
	allInt := true
	hasNull := false

	for i, v := range values {
		if naValues[v] {
			nulls[i] = true
			hasNull = true
			continue
		}
		parsed, err := strconv.ParseInt(v, 10, 64)
		if err != nil {
			allInt = false
			break
		}
		int64Vals[i] = parsed
	}

	if allInt && !hasNull {
		bldr := array.NewInt64Builder(alloc)
		defer bldr.Release()
		bldr.AppendValues(int64Vals, nil)
		return dtype.Int64, bldr.NewArray()
	}

	// Pass 2: try float64 (also handles the case where int64 works but has nulls)
	float64Vals := make([]float64, len(values))
	valid := make([]bool, len(values))
	allFloat := true

	for i, v := range values {
		if naValues[v] {
			valid[i] = false
			continue
		}
		parsed, err := strconv.ParseFloat(v, 64)
		if err != nil {
			allFloat = false
			break
		}
		float64Vals[i] = parsed
		valid[i] = true
	}

	if allFloat {
		bldr := array.NewFloat64Builder(alloc)
		defer bldr.Release()
		bldr.AppendValues(float64Vals, valid)
		return dtype.Float64, bldr.NewArray()
	}

	// Pass 3: fall back to string
	bldr := array.NewStringBuilder(alloc)
	defer bldr.Release()
	for i, v := range values {
		if naValues[v] {
			bldr.AppendNull()
		} else {
			bldr.Append(v)
		}
	}
	return dtype.String, bldr.NewArray()
}

// buildColumnWithDType forces parsing a column to a specific dtype.
func buildColumnWithDType(alloc memory.Allocator, values []string, dt dtype.DType, naValues map[string]bool) (arrow.Array, error) {
	switch dt {
	case dtype.Int64:
		bldr := array.NewInt64Builder(alloc)
		defer bldr.Release()
		for _, v := range values {
			if naValues[v] {
				bldr.AppendNull()
				continue
			}
			parsed, err := strconv.ParseInt(v, 10, 64)
			if err != nil {
				return nil, fmt.Errorf("cannot parse %q as int64: %w", v, err)
			}
			bldr.Append(parsed)
		}
		return bldr.NewArray(), nil

	case dtype.Float64:
		bldr := array.NewFloat64Builder(alloc)
		defer bldr.Release()
		for _, v := range values {
			if naValues[v] {
				bldr.AppendNull()
				continue
			}
			parsed, err := strconv.ParseFloat(v, 64)
			if err != nil {
				return nil, fmt.Errorf("cannot parse %q as float64: %w", v, err)
			}
			bldr.Append(parsed)
		}
		return bldr.NewArray(), nil

	case dtype.String:
		bldr := array.NewStringBuilder(alloc)
		defer bldr.Release()
		for _, v := range values {
			if naValues[v] {
				bldr.AppendNull()
			} else {
				bldr.Append(v)
			}
		}
		return bldr.NewArray(), nil

	default:
		return nil, fmt.Errorf("unsupported dtype override: %v", dt)
	}
}
```

**Important design note:** The `inferColumn` function implements the type inference algorithm:

1. **Pass 1 — int64:** Try `strconv.ParseInt` on every non-NA value. If all succeed and there are no nulls, build an `Int64` array. (Int64 arrays in Arrow do not support nulls cleanly for our use case, so any null forces float64.)
2. **Pass 2 — float64:** Try `strconv.ParseFloat` on every non-NA value. If all succeed, build a `Float64` array with null bitmap.
3. **Pass 3 — string:** Fall back to a `String` array with null bitmap.

This matches pandas' behavior where a column of integers with any NA becomes float64.

```go
package csv

import (
	"encoding/csv"
	"fmt"
	"io"
	"strconv"

	"github.com/vchitepu/gopandas/lib/dataframe"
)

// ToCSV writes the DataFrame to w in CSV format.
// Use CSVOption functions to customize output (e.g., WithSep).
func ToCSV(df dataframe.DataFrame, w io.Writer, opts ...CSVOption) error {
	cfg := defaultConfig()
	for _, opt := range opts {
		opt(&cfg)
	}

	writer := csv.NewWriter(w)
	writer.Comma = cfg.sep

	columns := df.Columns()
	nRows, _ := df.Shape()

	// Write header
	if err := writer.Write(columns); err != nil {
		return fmt.Errorf("csv write header: %w", err)
	}

	// Write data rows
	record := make([]string, len(columns))
	for row := 0; row < nRows; row++ {
		for colIdx, colName := range columns {
			val, err := df.At(row, colName)
			if err != nil {
				return fmt.Errorf("csv write: row %d, col %q: %w", row, colName, err)
			}
			record[colIdx] = formatValue(val)
		}
		if err := writer.Write(record); err != nil {
			return fmt.Errorf("csv write row %d: %w", row, err)
		}
	}

	writer.Flush()
	return writer.Error()
}

// formatValue converts a value to its string representation for CSV output.
func formatValue(v any) string {
	if v == nil {
		return ""
	}
	switch val := v.(type) {
	case int64:
		return strconv.FormatInt(val, 10)
	case float64:
		return strconv.FormatFloat(val, 'f', -1, 64)
	case string:
		return val
	case bool:
		return strconv.FormatBool(val)
	default:
		return fmt.Sprintf("%v", val)
	}
}
```

**Note:** The `formatValue` function uses `strconv.FormatFloat` with `'f'` format and precision `-1` (shortest representation) to avoid unnecessary trailing zeros while preserving exact values.

```go
package json

import (
	"encoding/json"
	"fmt"
	"io"
	"sort"

	"github.com/vchitepu/gopandas/lib/dataframe"
)

// FromJSON reads JSON data from r and returns a DataFrame.
// The orient parameter controls the expected JSON structure.
func FromJSON(r io.Reader, orient JSONOrient) (dataframe.DataFrame, error) {
	data, err := io.ReadAll(r)
	if err != nil {
		return dataframe.DataFrame{}, fmt.Errorf("json read: %w", err)
	}

	switch orient {
	case OrientRecords:
		return fromRecords(data)
	case OrientColumns:
		return fromColumns(data)
	case OrientIndex:
		return fromIndex(data)
	default:
		return dataframe.DataFrame{}, fmt.Errorf("json: unknown orient %v", orient)
	}
}

// fromRecords parses JSON in records format: [{col: val, ...}, ...]
func fromRecords(data []byte) (dataframe.DataFrame, error) {
	var records []map[string]any
	if err := json.Unmarshal(data, &records); err != nil {
		return dataframe.DataFrame{}, fmt.Errorf("json records: %w", err)
	}

	if len(records) == 0 {
		return dataframe.New(map[string]any{})
	}

	return dataframe.FromRecords(records)
}

// fromColumns parses JSON in columns format: {col: [val, ...], ...}
func fromColumns(data []byte) (dataframe.DataFrame, error) {
	var raw map[string][]any
	if err := json.Unmarshal(data, &raw); err != nil {
		return dataframe.DataFrame{}, fmt.Errorf("json columns: %w", err)
	}

	if len(raw) == 0 {
		return dataframe.New(map[string]any{})
	}

	// Convert map[string][]any to records format for FromRecords
	// First, determine column names and row count
	colNames := make([]string, 0, len(raw))
	for k := range raw {
		colNames = append(colNames, k)
	}
	sort.Strings(colNames)

	nRows := 0
	for _, vals := range raw {
		if len(vals) > nRows {
			nRows = len(vals)
		}
	}

	records := make([]map[string]any, nRows)
	for i := 0; i < nRows; i++ {
		rec := make(map[string]any, len(colNames))
		for _, name := range colNames {
			vals := raw[name]
			if i < len(vals) {
				rec[name] = vals[i]
			} else {
				rec[name] = nil
			}
		}
		records[i] = rec
	}

	return dataframe.FromRecords(records)
}

// fromIndex parses JSON in index format: {idx: {col: val, ...}, ...}
func fromIndex(data []byte) (dataframe.DataFrame, error) {
	// Use json.Decoder with ordered keys
	var raw map[string]map[string]any
	if err := json.Unmarshal(data, &raw); err != nil {
		return dataframe.DataFrame{}, fmt.Errorf("json index: %w", err)
	}

	if len(raw) == 0 {
		return dataframe.New(map[string]any{})
	}

	// Sort index keys for deterministic order
	idxKeys := make([]string, 0, len(raw))
	for k := range raw {
		idxKeys = append(idxKeys, k)
	}
	sort.Strings(idxKeys)

	// Convert to records
	records := make([]map[string]any, len(idxKeys))
	for i, key := range idxKeys {
		records[i] = raw[key]
	}

	return dataframe.FromRecords(records)
}
```

**Design notes:**
- `encoding/json` unmarshals all JSON numbers to `float64` by default. This means integer-valued columns (`"age": 30`) will arrive as `float64(30)`. The `dataframe.FromRecords()` function is responsible for handling these `any` values appropriately.
- The `fromColumns` and `fromIndex` functions convert to records format and delegate to `dataframe.FromRecords()` for uniform construction.
- Column ordering in `fromColumns` uses `sort.Strings` since JSON maps have no guaranteed order.

```go
import (
	"encoding/json"
	"fmt"
	"io"
	"strconv"

	"github.com/vchitepu/gopandas/lib/dataframe"

// ToJSON writes the DataFrame to w in the specified JSON orient format.
func ToJSON(df dataframe.DataFrame, w io.Writer, orient JSONOrient) error {
	switch orient {
	case OrientRecords:
		return writeRecords(df, w)
	case OrientColumns:
		return writeColumns(df, w)
	case OrientIndex:
		return writeIndex(df, w)
	default:
		return fmt.Errorf("json: unknown orient %v", orient)
	}
}

// writeRecords writes JSON in records format: [{col: val}, ...]
func writeRecords(df dataframe.DataFrame, w io.Writer) error {
	nRows, _ := df.Shape()
	columns := df.Columns()

	records := make([]map[string]any, nRows)
	for i := 0; i < nRows; i++ {
		rec := make(map[string]any, len(columns))
		for _, col := range columns {
			val, err := df.At(i, col)
			if err != nil {
				return fmt.Errorf("json write: row %d, col %q: %w", i, col, err)
			}
			rec[col] = val
		}
		records[i] = rec
	}

	enc := json.NewEncoder(w)
	enc.SetIndent("", "  ")
	return enc.Encode(records)
}

// writeColumns writes JSON in columns format: {col: [val, ...], ...}
func writeColumns(df dataframe.DataFrame, w io.Writer) error {
	nRows, _ := df.Shape()
	columns := df.Columns()

	result := make(map[string][]any, len(columns))
	for _, col := range columns {
		vals := make([]any, nRows)
		for i := 0; i < nRows; i++ {
			val, err := df.At(i, col)
			if err != nil {
				return fmt.Errorf("json write: row %d, col %q: %w", i, col, err)
			}
			vals[i] = val
		}
		result[col] = vals
	}

	enc := json.NewEncoder(w)
	enc.SetIndent("", "  ")
	return enc.Encode(result)
}

// writeIndex writes JSON in index format: {idx: {col: val, ...}, ...}
func writeIndex(df dataframe.DataFrame, w io.Writer) error {
	nRows, _ := df.Shape()
	columns := df.Columns()
	idx := df.Index()
	labels := idx.Labels()

	result := make(map[string]map[string]any, nRows)
	for i := 0; i < nRows; i++ {
		rec := make(map[string]any, len(columns))
		for _, col := range columns {
			val, err := df.At(i, col)
			if err != nil {
				return fmt.Errorf("json write: row %d, col %q: %w", i, col, err)
			}
			rec[col] = val
		}
		// Use index label as the key
		key := formatIndexLabel(labels[i])
		result[key] = rec
	}

	enc := json.NewEncoder(w)
	enc.SetIndent("", "  ")
	return enc.Encode(result)
}

// formatIndexLabel converts an index label to a string key.
func formatIndexLabel(label any) string {
	switch v := label.(type) {
	case string:
		return v
	case int64:
		return strconv.FormatInt(v, 10)
	case int:
		return strconv.Itoa(v)
	case float64:
		return strconv.FormatFloat(v, 'f', -1, 64)
	default:
		return fmt.Sprintf("%v", v)
	}
}
```

```go
package parquet

import (
	"context"
	"fmt"
	"io"

	"github.com/apache/arrow-go/v18/arrow"
	"github.com/apache/arrow-go/v18/arrow/array"
	"github.com/apache/arrow-go/v18/arrow/memory"
	"github.com/apache/arrow-go/v18/parquet/file"
	"github.com/apache/arrow-go/v18/parquet/pqarrow"
	"github.com/vchitepu/gopandas/lib/dataframe"
)

// FromParquet reads a Parquet file and returns a DataFrame.
// The reader must implement io.ReaderAt (e.g., *os.File, *bytes.Reader).
// size is the total size of the Parquet data in bytes.
func FromParquet(r io.ReaderAt, size int64) (dataframe.DataFrame, error) {
	pf, err := file.NewParquetReader(r, file.WithReadProps(file.ReaderProperties{
		BufferedStreamEnabled: true,
	}))
	if err != nil {
		return dataframe.DataFrame{}, fmt.Errorf("parquet open: %w", err)
	}
	defer pf.Close()

	arrowReader, err := pqarrow.NewFileReader(pf, pqarrow.ArrowReadProperties{
		Parallel:  false,
		BatchSize: 0,
	}, memory.DefaultAllocator)
	if err != nil {
		return dataframe.DataFrame{}, fmt.Errorf("parquet arrow reader: %w", err)
	}

	tbl, err := arrowReader.ReadTable(context.Background())
	if err != nil {
		return dataframe.DataFrame{}, fmt.Errorf("parquet read table: %w", err)
	}
	defer tbl.Release()

	tr := array.NewTableReader(tbl, tbl.NumRows())
	defer tr.Release()

	if !tr.Next() {
		// Empty table — build empty record preserving schema
		schema := tbl.Schema()
		emptyArrays := make([]arrow.Array, schema.NumFields())
		for i, field := range schema.Fields() {
			bldr := array.NewBuilder(memory.DefaultAllocator, field.Type)
			emptyArrays[i] = bldr.NewArray()
			bldr.Release()
		}
		rec := array.NewRecord(schema, emptyArrays, 0)
		for _, a := range emptyArrays {
			a.Release()
		}
		defer rec.Release()
		return dataframe.FromArrow(rec), nil
	}

	rec := tr.Record()
	rec.Retain()
	defer rec.Release()

	return dataframe.FromArrow(rec), nil
}
```

```go
package parquet

import (
	"fmt"
	"io"

	"github.com/apache/arrow-go/v18/arrow"
	"github.com/apache/arrow-go/v18/arrow/array"
	"github.com/apache/arrow-go/v18/parquet/pqarrow"
	"github.com/vchitepu/gopandas/lib/dataframe"
	"github.com/vchitepu/gopandas/lib/dtype"
)

// ToParquet writes the DataFrame to w in Parquet format.
func ToParquet(df dataframe.DataFrame, w io.Writer) error {
	nRows, _ := df.Shape()
	columns := df.Columns()
	dtypes := df.DTypes()

	// Build Arrow schema from DataFrame columns
	fields := make([]arrow.Field, len(columns))
	for i, name := range columns {
		arrowDT, err := dtype.DTypeToArrow(dtypes[i])
		if err != nil {
			return fmt.Errorf("parquet write: column %q: %w", name, err)
		}
		fields[i] = arrow.Field{Name: name, Type: arrowDT, Nullable: true}
	}
	schema := arrow.NewSchema(fields, nil)

	// Build Arrow arrays from DataFrame columns
	arrays := make([]arrow.Array, len(columns))
	for i, name := range columns {
		col, err := df.Col(name)
		if err != nil {
			return fmt.Errorf("parquet write: %w", err)
		}
		// Get the underlying Arrow array from the Series
		arr := col.Array()
		arrays[i] = arr
		arrays[i].Retain() // retain because we'll release later
	}

	// Build record
	rec := array.NewRecord(schema, arrays, int64(nRows))
	defer rec.Release()
	for _, arr := range arrays {
		arr.Release()
	}

	// Write Parquet
	writer, err := pqarrow.NewFileWriter(schema, w, nil, pqarrow.DefaultWriterProps())
	if err != nil {
		return fmt.Errorf("parquet writer create: %w", err)
	}

	if err := writer.Write(rec); err != nil {
		return fmt.Errorf("parquet write record: %w", err)
	}

	if err := writer.Close(); err != nil {
		return fmt.Errorf("parquet writer close: %w", err)
	}

	return nil
}
```

## Implementation Notes

### Type Inference Algorithm (CSV)

The CSV reader's `inferColumn` function implements a three-pass type inference:

1. **Pass 1 — int64:** Try `strconv.ParseInt` on every non-NA value. If all succeed **and there are no nulls**, build an `Int64` array. (Arrow Int64 arrays handle nulls, but matching pandas behavior: any null in an integer column promotes to float64.)
2. **Pass 2 — float64:** Try `strconv.ParseFloat` on every non-NA value. If all succeed, build a `Float64` array with a null bitmap for NA values.
3. **Pass 3 — string:** Fall back to a `String` array with a null bitmap.

This matches pandas' behavior: `pd.read_csv` with a column of `[1, 2, NA]` produces `float64`, not `int64`.

### JSON Number Handling

Go's `encoding/json` unmarshals JSON numbers to `float64` by default. This means:
- `{"age": 30}` produces `float64(30)`, not `int64(30)`
- The `dataframe.FromRecords()` function receives `float64` values for all numeric JSON fields
- This is intentional and matches the behavior of most JSON parsers

### Parquet Arrow Bridge

The Parquet reader/writer uses the Arrow ecosystem directly:
- **Read:** Parquet file → `pqarrow.FileReader` → Arrow `Table` → Arrow `Record` → `dataframe.FromArrow()`
- **Write:** `DataFrame.Col()` → `Series.Array()` → Arrow `Record` → `pqarrow.FileWriter` → Parquet file

This avoids any intermediate data copies — the Arrow arrays from Parquet are used directly as the backing storage for Series.
